# 💳 Credit Card Fraud Detection

A Machine Learning project that predicts whether a credit card transaction is **fraudulent or legitimate** based on various transaction features.

This project demonstrates how classification algorithms can be used to detect fraudulent activities and improve financial security systems.

---

# 🧠 About the Project

Credit Card Fraud Detection is a **binary classification problem** where the model predicts:

- **1 → Fraudulent Transaction**
- **0 → Legitimate Transaction**

Using historical transaction data, the machine learning model learns patterns associated with fraudulent activities and helps identify suspicious transactions automatically.

---

# 🚀 Features

- Perform data preprocessing
- Exploratory Data Analysis (EDA)
- Detect fraudulent transactions
- Train classification models
- Evaluate model performance
- Visualize fraud detection results
- Beginner-friendly implementation
